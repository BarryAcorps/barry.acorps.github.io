Hello World

April 28, 2024
Tags:

Hello everyone. This is the first ever blog post of the ‘barry.acorps’ website. It took me a while to set up, but I’m very proud of what I’ve done so far with Hugo.

So, what is this website?

I’ve set up this website to pretty much act as my platform, my portfolio, and also to put my own tutorials in word form. This is also where I’ll be putting my notes for other stuff secretly.

This website uses “markdown” format so typing these blogs are going to be like writing EXEMPLAR update logs. Speaking of which, I will be putting my update logs here instead once I figure out how to put discord embeds into my website.

This formatting actually has a lot of functionality. First, there’s image support, which you can see in the works section. Then there’s also MathJax formatting for mathematical equations.

For example…

The magnitude of electrostatic force between two charged particles is

\[ F_e = \frac{1}{4\pi\epsilon_0}\frac{q_0q_1}{r^2} \]

Where \(\epsilon_0\) is the permittivity of free space. \(q_0\) and \(q_1\) are the amount of charge of the two particles, and \(r\) is the distance between the two particles.

How did you make this website?

Interestingly enough, I didn’t make this out of scratch. I’m using Hugo, a static-site generator that can help me manage my content here. It’s like Wordpress, but more technical. Actually, I don’t know, because I’ve never used Wordpress, ever. Still, you need to have some HTML expertise along with some CSS. Hugo just does all of the automating for you.

You can still see vestiges of it if you Ctrl + U to reveal the source code in the website. The whitespaces are all wonky and strange. That’s because I haven’t figured out how to deal with the whitespaces when it comes to the code that automates making the pages.
